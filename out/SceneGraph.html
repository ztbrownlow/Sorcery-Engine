Namespace: SceneGraph

SceneGraph

A SceneGraph holds all of the objects in the game
Properties:
Name Type Description
name string the name of the scenegraph
doUpdate boolean the value of if this scenegraph should update. Default value is true.
doDraw boolean the value of if objects in this scenegraph should be drawn. Default value is true.
isClickable boolean the value of if the objects in the scenegraph should be clickable. Default value is true.
children array the array that holds all the objects
updateDirection string the direction that the objects will update in. Defaults to "forwards". This value can only be either "forwards" or "backwards".
drawDirection string the direction that the objects will be drawn. Defaults to "backwards". This value can only be either "forwards" or "backwards"
isSceneGraph boolean returns true if the object is a scenegraph. Default true.
